Welcome to a game of colors.

How to Play:

The game features a big board that displays a target color

Below the target color, you'll see six tabs in a separate board

Each tab is a button, and an option

You are to choose the tab with a color that matches the exact target color

You are awarded a point for every correct match

You can stretch the options container to have a better view of the colors
